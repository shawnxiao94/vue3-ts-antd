# vue3-typescript-antd

> 基于@vue/cli 4.5.9 搭建的vue3 + typescript + antd项目

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
